# Woodstove1 - 3D Модель дров'яної печі

## Опис

Інтерактивна 3D-модель дров'яної печі, створена з використанням Three.js. Проєкт дозволяє в реальному часі змінювати розміри та параметри печі і бачити зміни в 3D.

## Можливості

- 🔥 Реалістична 3D-модель дров'яної печі
- 🧱 Базова архітектура з проєктними габаритами 600×600×1200 мм
- 🏗️ Окремі об'єкти: корпус, внутрішня камера, бафль, димохід, дверця зі склом, ручка, ніжки
- 🛡️ Окремі геометрії для сталевого шару та шамотної облицювальної цегли
- 🧱 Корпус з окремих сталевих мешів: дно, бокові стінки, задня стінка, верхня кришка
- 🔥 Футерівка шамотом реалізована як окремий шар панелей з урахуванням товщини
- 🦿 Ніжки/опори є окремими об’єктами і прив’язані до корпусу
- 🎛️ Параметризація: товщина сталі (3-5 мм), шамот (50 мм), дверця/скло, димохід (150-200 мм)
- 🕹️ Регулювання бафля: висота, кут 45°-90° і боковий важіль справа пічки
- ↕️ Baffle зв'язаний з конфігом: висота установки, кут, передній/задній зазори (через UI + Apply)
- 🌬️ Передня пластина первинного повітря з овальними отворами 50×35 мм (кількість/координати параметризовані)
- 🧰 Окрема задвижка первинного повітря з параметром відсотка перекриття отворів
- 💨 Доданий air wash канал перед склом (над дверцятами) з параметром щілини та візуальною анімацією відкриття
- 🧭 Режими камери: 3D view / Drawing front / Drawing side / Drawing top (фіксовані ракурси для креслень)
- 📐 Drawing mode з розмірними лініями та підписами (frontView / sideView / topView дані)
- 👁️ Тогли видимості: шамот, бафль, повітряні канали, димохід
- 🧨 Explode view з параметрами offset для вузлів (front/back/top/baffle/brick lining) та анімацією рознесення/повернення
- 🔥 Набір пресетів режимів: start-up / low / medium / high / overnight (primary/secondary air, air wash, flame intensity)
- 🔥 Спрощена візуалізація полум'я в камері (particle-like), пов'язана з інтенсивністю/висотою/кольором полум'я
- 🌡️ Спрощений тепловий градієнт корпусу (холодний/теплий/гарячий) залежно від режиму
- 📊 Додано розрахунок орієнтовних технічних метрик за режимом: КПД, теплова віддача (кВт) та час горіння від об'єму закладки
- ⚠️ Додано правила попереджень/hints у UI для небезпечних режимів повітря (ризик диму/недогорання та надто гарячий режим)
- 🧠 Розрахунки винесено в модуль `PhysicsModel` (КПД, кВт, час горіння, умовний потік повітря, температура, warnings)
- 🧼 UI панелі згруповано у чисті секції (конфіг, режими/метрики/видимість, швидкі параметри, сервісні дії)
- ❓ Додано Tour/Help overlay з поясненнями: baffle, air wash, режими, drawing mode, explode view
- 💾 Додано експорт/імпорт конфігурації у JSON для збереження та завантаження варіантів пічки
- 🔴 Додано видимий маркер-об'єкт у сцені для швидкої візуальної перевірки
- ↪️ Додано `404.html` з редіректом на `index.html`, щоб прибрати сценарії `Not Found` у прев'ю
- 🎥 Сцена з OrbitControls (обертання, панорамування, зум)
- 🧭 Додані координатні осі та сітка з можливістю вмикати/вимикати
- 📐 Масштаб сцени зафіксовано в коді: 1 одиниця = 1 см
- 🧩 Окремий конфіг-об'єкт пічки (габарити, камера, сталь/шамот, димохід, дверця, скло)
- 📝 Проста форма редагування конфігу з кнопкою `Apply` і перегенерацією геометрії
- 🚪 Дверця як окремий вузол (рамка + скло + вісь), анімація відкриття/закриття через клік і слайдер
- 📏 Налаштування розмірів:
  - Ширина (30-100 см)
  - Глибина (30-100 см)
  - Висота (50-150 см)
  - Висота ніжок (0-50 см)
- 🖱️ Обертання камери мишею
- 🔍 Масштабування колесом миші
- 📱 Адаптивний дизайн

## Технології

- HTML5
- CSS3
- JavaScript
- Three.js (r128)

## Використання

1. Відкрийте `index.html` у браузері
2. Використовуйте слайдери зліва для зміни параметрів печі
3. Натисніть і перетягуйте мишею для обертання камери
4. Використовуйте колесо миші для масштабування
